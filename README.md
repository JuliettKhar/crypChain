# crypChain
This course is a remastered version of the original Build a Blockchain and Cryptocurrency from Scratch course released in early 2018: https://www.udemy.com/build-blockchain.

A simple blockchain implementation in TypeScript to explore cryptographic hashing and chain validation.

## About

CrypChain demonstrates the basic backbone of a blockchain:
- block creation
- hashing with SHA-256
- linking blocks
- integrity validation

Designed as a learning project to understand decentralized ledger fundamentals.
